# Error Enum

Used to generate documentation comments and `Display` implementation for tree-structured error types.

## Concepts

|    Concept     |            Example             |
| :------------: | :----------------------------: |
|     Number     |             `1234`             |
|      Code      |            `E1234`             |
|      Kind      |            `error`             |
|  Kind Acronym  |              `E`               |
|      Kind      |         `error[E1234]`         |
| Message Prefix |        `error[E1234]: `        |
|  Description   |        `Access denied.`        |
|    Message     | `error[E1234]: Access denied.` |
